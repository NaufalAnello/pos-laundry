Pelanggan

Jika tombol-tombol di atas terlihat, berarti kode HTML sudah benar.

Masalah kemungkinan ada di browser cache atau server belum restart.

Cara Fix:

  1. Hard Refresh Browser: Ctrl + Shift + R (Windows/Linux) atau Cmd + Shift + R (Mac)
  2. Clear Browser Cache: Ctrl + Shift + Delete
  3. Restart Server: pm2 restart pos-laundry
  4. Cek di Incognito Mode

Verifikasi di Browser:

  1. Buka halaman pelanggan: http://localhost:3001/pages/pelanggan.html
  2. Tekan F12 untuk buka DevTools
  3. Klik tab "Elements"
  4. Cari elemen dengan class "page-header"
  5. Lihat apakah ada 4 tombol di dalamnya

Check via Console:

// Di browser console, jalankan:
document.querySelectorAll('.page-header button').length
// Harus return: 4

document.querySelector('.page-header').innerHTML
// Harus menampilkan 4 button tags
