📋 Panduan Akses USB di Linux

Jika muncul error EACCES: permission denied saat cetak:

  1. Cek device: ls -la /dev/usb/
  2. Tambah user ke grup: sudo usermod -a -G lp $USER
  3. Logout dan login ulang, atau: newgrp lp
  4. Alternatif sementara: sudo chmod 666 /dev/usb/lp0

💾 Backup & Restore Database
⬇️

Download Backup

Download salinan database SQLite. Simpan di tempat aman sebagai cadangan data.

⬆️

Restore dari Backup

Upload file backup .db. File akan disimpan sementara. Server perlu direstart untuk menerapkan restore.

⚠️ Perhatian Restore:
1. Download backup terbaru sebelum restore.
2. Upload file .db backup yang valid.
3. Setelah upload berhasil, hentikan server dan rename file laundry.db.restore ke laundry.db.
4. Jalankan ulang server.
