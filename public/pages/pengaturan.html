📋 Panduan Akses USB di Linux

Jika muncul error EACCES: permission denied saat cetak:

  1. Cek device: ls -la /dev/usb/
  2. Tambah user ke grup: sudo usermod -a -G lp $USER
  3. Logout dan login ulang, atau: newgrp lp
  4. Alternatif sementara: sudo chmod 666 /dev/usb/lp0
